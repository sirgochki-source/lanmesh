# GUI Phase 1 — сквозная проверка (Command Glass)

Проверка выполнена в браузере через **headless Chrome (Chromium — тот же движок,
что WebView2)**: точный вьюпорт форсируется через CDP `Emulation.setDeviceMetricsOverride`,
скриншоты — `Page.captureScreenshot`, интерактив — `Runtime.evaluate` + клики.
Дев-инструменты в `scratchpad/*.mjs` (shot/nav/click/toast/inv/set-check), не в репозитории.
Go на машине не установлен — весь фронтенд проверялся так, без Go (по стратегии спеки §11).

Дата: 2026-07-19. Ветка `gui-redesign`.

## Компактный режим (истинные 380px, `overflowX:false` во всех сценариях)
- [x] `disconnected` — pill «не подключено» (off), self-строки нет, форма «＋ Добавить сеть» раскрыта (онбординг).
- [x] `empty` — карточка сети + подсказка «Позови друга…».
- [x] `team3` — 4 участника; точки-статусы (мятный=direct, синий=relay, жёлтый=connecting); пинги окрашены по порогам (22 мятный / 125 жёлтый / 48 мятный / «подключение…»); self-строка «твой IP · внешний · аптайм 2 ч 14 м».
- [x] `multi` — pill «2 сети», несколько карточек сетей.
- [x] `noext` — pill «ограничено», warn-бокс «Внешний адрес неизвестен».
- [x] `sigerr` — pill «ограничено», жёлтый индикатор «🟡 myteam» (проблема сигналки).
- [x] `hostile` — имя `a‹U+202E›gnp.exe` отрисовано как **`agnp.exe`** (bidi-override вычищен, БЕЗ разворота текста) — санитизация подтверждена.

## Подробный режим (1000px → detailed по ширине)
- [x] `team3` — рейл (бренд, сеть, навигация Список/Карта/Трафик/Настройки), 3 плитки (Качество «среднее» 2 direct·1 relay / Трафик «Phase 3» / Участников 4), богатые строки (аватар-инициал, сигнал-бары, sparkline пинга, бейдж, пинг). Дубля вордмарка нет (бренд только в рейле).
- [x] `multi` — рейл показывает обе сети (myteam/dota), активная подсвечена.
- [x] Вид «Настройки» — секция СЕРВЕРЫ (сигналки-textarea + relay-input, `disabled` при подключённой сети, `opacity:0.5` подтверждён CDP-замером; адреса серверов не показаны) + ДИАГНОСТИКА (Отправить диагностику + автоотправка логов).
- [x] Виды «Карта»/«Трафик» — заглушки «скоро» (Phase 3/4).

## Живые фичи и интерактив (CDP)
- [x] **Sparkline пинга** — за несколько опросов линии оживают; цвет по качеству (мятный direct, янтарный relay); у `connecting` (нет RTT) графика нет.
- [x] **Тосты вход/выход** — живое переключение multi→team3 дало `["sara_pc в сети","Dev_null в сети","gamer2000 вышел"]`; первый снимок не тостится.
- [x] **Копирование IP** — клик по IP → чип «IP 25.44.12.9 скопирован» (в WebView2 пишется и буфер).
- [x] **Переключение сети** — клик по «dota» в рейле → основная панель myteam→dota (заголовок и участник сменились).
- [x] **Приглашение** — клик → `/api/invite` → фидбек «✓ скопировано» (устойчиво к сбою буфера).
- [x] **Отзывчивость** — `rootMode` следует ширине окна (compact <620px, detailed ≥620px).
